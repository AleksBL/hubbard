DOS_distribution¶

class hubbard.plot.DOS_distribution(HH, PDOS, sites=[], ext_geom=None, realspace=False, **kwargs)[source]¶

Bases: hubbard.plot.plot.GeometryPlot

Plot LDOS in the configuration space for the hubbard.HubbardHamiltonian object

Parameters

  • HH (HubbardHamiltonian) – mean-field Hubbard Hamiltonian

  • PDOS (numpy.ndarray) – Density of states projected in atomic sites obtained at certain energy

  • realspace – If True it will plot the DOS in a realspace grid otherwise it plots it as a scatter plot with varying size depending on the PDOS numerical value
